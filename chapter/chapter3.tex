\chapter{系统配置}
正确编译需要以下几个部分(这是一个列表环境）：
\begin{itemize}
\item 一个基本的TEX发行版
\item CJK或XeCJK（供\LaTeX{}）宏包
\item ctex宏包（提供ctexbook文档).
\item 中文字体
\item 如果要使用 biblatex 进行文献列表和引用的排版的话， 还需要 biblatex 宏包。
\end{itemize}
